A customizable grid block lets users create Bento Layout in block editor.

== Description ==
Bento Box is a customizable WordPress block plugin that lets users create **Bento layouts** or **simple grids** inside Gutenberg.

**Key Features:**
- **Random Layout Generation:**  Automatically generates a unique Bento pattern.
- **Hover Focus Effect:** Highlights the hovered card while blurring others.
- **Customizable Backgrounds & Hover Effects:** Change background colors, images, and effects.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/bento-box` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
